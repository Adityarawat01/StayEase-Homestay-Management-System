# StayEase - Eco-Friendly Homestay Booking Platform

StayEase is a frontend web application built with React and Vite. It allows travelers to discover eco-friendly homestays and unique stays while supporting local communities.

## Features

- Browse and filter a curated list of eco-friendly properties
- View detailed property information
- Interactive Host Dashboard for managing listings and booking requests
- Mobile-responsive design

## Tech Stack

- React
- Vite
- React Router
- Vanilla CSS

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the application in your web browser (usually `http://localhost:5173`)
